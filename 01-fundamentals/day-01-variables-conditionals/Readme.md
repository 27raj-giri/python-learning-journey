# Day 1 - Variables & Conditionals

**Date:** December 15, 2025  
**Programs:** 8  
**Time:** 3 hours  
**Focus:** Python basics, input/output, conditionals

## Topics Covered
- Variables (int, float, string)
- Type casting
- User input
- Conditional statements (if/else/elif)
- Comparison operators
- Modulo operator

## Programs
1. square_area.py - Calculate area of square
2. average_calculator.py - Average of two numbers
3. number_comparison.py - Compare two numbers (>=)
4. even_odd_checker.py - Check if even or odd
5. greatest_of_three.py - Find largest of three numbers
6. hello_world.py - Print name
7. multiple_of_five.py - Check divisibility by 5
8. sum_calculator.py - Add two numbers

## Key Learnings
- Input is always string - need type casting
- Modulo (%) for divisibility checks
- elif is more efficient than multiple ifs
