# Day 8 - Dictionaries Introduction

**Date:** December 22, 2025  
**Programs:** 7  
**Time:** 4 hours  
**Focus:** Dictionaries, sets, counting patterns

## Topics Covered
- Dictionary basics (key-value pairs)
- dict.get() method
- Counting with dictionaries
- Iterating over dict.items()
- Set basics
- Dictionary filtering

## Programs
1. anagram_checker.py - Check if two strings are anagrams
2. find_duplicates.py - Find duplicate numbers in list
3. email_domain_counter.py - Count email domains
4. set_demo.py - Demonstrate set behavior
5. dict_filter_values.py - Filter dict by values
6. group_by_length.py - Group words by length
7. character_frequency.py - Count character occurrences

## Key Learnings
- Dictionaries are powerful for counting
- dict.get() avoids KeyError
- Can iterate over keys, values, or both
- Sets automatically remove duplicates
